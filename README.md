# CENTRAL-HEX- (IB_HEX_BOT)

Bot WhatsApp en français basé sur Baileys, avec:
- Préfixe obligatoire: Ib
- Mode privé (owner only)
- Page Web QR Code (Render)
- 200 commandes exactement

## Déploiement sur Render
1. Push sur GitHub
2. Render > New > Web Service > Connect repo
3. Build: npm install
4. Start: npm start

## Connexion WhatsApp
Ouvre l’URL Render de ton service: la page affichera le QR Code.
Scanne dans WhatsApp > Appareils connectés.

## Exemple
- Ibmenu
- Iballcmds
- Ibping
